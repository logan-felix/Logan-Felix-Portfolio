IntelliJ Projects

This was the third project for my Object Oriented Programming class. It was interesting for me because it involved inheritance from one class to many sub-classes. It allowed me to practice using inheritance a lot.

For the fourth project of my Object Oriented Programming, the task was to create a zombie generator, which involved several types of zombies that inherited from the abstract Zombie class. I learned the importance and how to use abstract classes in Java through this project.

The fifth project was to make a code scrambler, which took in a String from the user and encrypted/decrypted it depending on what was selected by the user. I practiced using interfaces in this project with an interface called IStringEncryptable that was implemented by encryptor classes, such as Doubler, Cutter, and VowelReplacer. I enjoyed making the logic and learning about how interfaces work.

Visual Studio Code Projects

I used Visual Studio Code to make several websites and projects for my Intro to Web class. I learned the basics of HTML, CSS, and JavaScript. In this project, I made a number guessing game. The JS is used to generate a random number. It then takes any guesses from the user in the text box upon the guess button being clicked. It then lets the user know if the number is higher or lower than their guess. It congratulates the user when they get the number and says how many guesses it took.

In this project, I used HTML and CSS to make a website about myself. I included my interests, favorite sites, and future plans. It allowed me to get good practice with using HTML and CSS.
